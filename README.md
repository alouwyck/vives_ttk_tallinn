**Guest Lectures at Centre for Sciences, TTK University of Applied Sciences, Tallinn, Estonia**

by Andy Louwyck & Dominique Stove - VIVES University of Applied Sciences, Kortrijk, Belgium 

8 - 12 December 2025

Overview:

* The [hydro](https://github.com/alouwyck/vives_ttk_tallinn/tree/main/hydro) folder contains an introduction to hydrogeology and groundwater modeling, which is part of the Environmental Physics course.
* The [oop_python](https://github.com/alouwyck/vives_ttk_tallinn/tree/main/oop_python) folder contains an introduction to object-oriented programming in Python, which is part of the Introduction to Python course.
* The [numpy](https://github.com/alouwyck/vives_ttk_tallinn/tree/main/numpy) folder contains an introduction to NumPy, which is part of the Introduction to MATLAB course.
* The [intro_ml](https://github.com/alouwyck/vives_ttk_tallinn/tree/main/intro_ml) folder contains an introduction to machine learning, which is part of the Statistics course.
* The [intro_dl](https://github.com/alouwyck/vives_ttk_tallinn/tree/main/intro_dl) folder contains an introduction to deep learning, which is part of the Applied Mathematics course.
